# UI Anti Patterns

Avoid the following patterns unless absolutely necessary.

## Excessive decoration

Avoid:

- heavy gradients
- excessive shadows
- glassmorphism
- neumorphism

---

## Visual clutter

Avoid:

- too many cards
- unnecessary icons
- multiple accent colors
- decorative elements without purpose

---

## Overcomplicated layouts

Avoid:

- deeply nested containers
- crowded dashboards
- too many UI sections

Prefer calm and simple layouts.
